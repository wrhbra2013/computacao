Crebortoli - Site Institucional

O projeto Crebortoli consiste em um site institucional completo desenvolvido para apresentar os serviços, portfólio e informações de contato de uma empresa do setor de beleza e cuidados pessoais. O site foi desenvolvido utilizando HTML, CSS e JavaScript vanilla, sem dependências de frameworks, garantindo carregamento rápido e fácil manutenção.

A estrutura do site inclui múltiplas páginas como home, serviços, portfólio, sobre a empresa, área do cliente e documentação técnica. O design segue uma identidade visual moderna com cores vibrantes (rosa e tons claros), tipografia elegante e layout responsivo que se adapta a diferentes tamanhos de tela, desde dispositivos móveis até desktops.

Entre as funcionalidades principais destacam-se um sistema de agendamento de serviços online que permite aos clientes escolherem data, horário e profissional desejado, confirmações automáticas por WhatsApp, gestão de clientes com histórico de atendimentos, e um sistema de login para clientes acompanharem seus agendamentos. O site também integra com APIs de terceiros para envio de mensagens e notificações.

O projeto foi desenvolvido com foco em experiência do usuário e convertibilidade, incluindo formulários de contato otimizados, chamadas para ação estratégicas e integração com redes sociais. A documentação técnica detalhada foi criada para facilitar futuras manutenções e expansões do sistema, incluindo diagramas de arquitetura, descrição de endpoints da API e instruções de deployment.

Projetos Dinâmicos - Portfólio Web

O projeto Projetos Dinâmicos é um portfólio web abrangente que demonstra diversas técnicas e tecnologias de desenvolvimento frontend e backend. O projeto foi construí­do utilizando uma combinação de tecnologias modernas, incluindo HTML5, CSS3, JavaScript vanilla e Node.js, showcasing a amplitude de habilidades em desenvolvimento web fullstack.

O site principal apresenta uma interface altamente interativa com animações fluidas, transições suaves entre secciones e efeitos visuais que demonstram domínio de CSS3 e JavaScript. Os projetos incluem desde calculators e conversores até jogos interativos, aplicações de drawing canvas, geradores de conteúdo dinâmico e integrações com APIs externas.

Entre os projetos destacados estão uma aplicação de gestão financeira pessoal com gráficos e proyecciones, um sistema de notas com persistência local, geradores de senhas seguras, ferramentas de conversão de unidades, jogos de tabuleiro virtuais, e demos de manipulação de canvas HTML5. Cada projeto foi desenvolvido com código limpo, comments explicativos e estrutura modular.

O repositório também inclui versões desktop do portfólio desenvolvidas com Tauri (Rust + WebView), demonstrando capacidade de criar aplicações nativas para Windows utilizando tecnologias web. A documentação inclui setup de ambiente, estrutura de arquivos, descrições de cada projetoIndividual e instruções para contribuições.
