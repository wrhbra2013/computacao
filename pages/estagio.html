Projeto Amor Animal Marília

Sobre a ONG

O Amor Animal Marília é uma organização não-governamental dedicada à proteção e bem-estar animal no município de Marília-SP. A ONG realiza resgate, tratamento, castração e adoção de animais abandonados, trabalhando com voluntários e doações da comunidade. A instituição atua há anos na região, promovendo campanhas de conscientização e oferecendo suporte aos animais em situação de vulnerabilidade.

Descrição do Projeto

O sistema web completo para gestão da ONG foi desenvolvido como parte das atividades do curso de Engenharia da Computação da UNIVESP. O objetivo principal foi criar uma plataforma digital que substituísse os processos manuais e planilhas utilizadas pela organização, permitindo um gerenciamento mais eficiente e profissional de todas as áreas de atuação da ONG.

A aplicação foi construída utilizando Node.js como runtime JavaScript no backend, Express como framework web, PostgreSQL para armazenamento de dados e templates EJS para renderização das views. O sistema contempla funcionalidades para cadastro e gerenciamento de animais, controle completo do fluxo de adoções, gerenciamento de voluntários e suas respectivas escalas, além de módulos para controle de castrações e parcerias com clínicas veterinárias.

Funcionalidades Principais

O sistema possui múltiplos módulos que cobrem todas as necessidades operacionais da ONG. O módulo de gestão de animais permite cadastrar novos animais com fotos, histórico médico e situação atual, facilitando o acompanhamento de cada caso. O controle de adoções gerencia todo o processo desde o interesse inicial até a finalização do contrato, incluindo verificação de requisitos e acompanhamento pósadoção.

O módulo de voluntários permite cadastrar membros da equipe, registrar suas disponibilidade e gerenciar escalas de trabalho. O sistema de transparência publica automaticamente informações sobre receitas e despesas, promovendo uma gestão mais aberta e confiável. Há também módulos específicos para eventos, parcerias com clínicas, campanhas de castração e um sistema de busca de animais perdidos.

Tecnologias Utilizadas

O backend foi desenvolvido em Node.js utilizando o framework Express, com中间件 para manipulação de sessões, uploads de arquivos e flash messages. O banco de dados PostgreSQL armazena todas as informações de forma estruturada, com tabelas para animais, adoções, voluntários, eventos, transparência e controle de acesso. O frontend utiliza HTML, CSS e JavaScript com templates EJS para renderização server-side.

A aplicação seguiu boas práticas de desenvolvimento como separação de responsabilidades em rotas, controladores e modelo de dados, autenticação baseada em sessões, tratamento de erros estruturado e código organizado em camadas. O sistema também inclui funcionalidades de segurança como proteção contra SQL injection, validação de dados e controle de acesso por níveis de permissão.

Resultado e Impacto

Com a implementação do sistema, a ONG passou a contar com uma ferramenta profesional para gestão de suas atividades diárias. O controle automatizado reduziu significativamente o tempo gasto em tarefas administrativas, permitindo que os voluntários se dedicassem mais às atividades de campo. A transparência nas informações contribuiu para aumentar a confiança dos doadores e parceiros.

O projeto também representou uma experiência de aprendizado significativa no contexto do curso de Engenharia da Computação, permitindo a aplicação prática de conhecimentos em desenvolvimento web fullstack, banco de dados, arquitetura de software e metodologias de desenvolvimento. A complexidade do sistema challenges abrangendo desde o desenvolvimento backend até a implementação de interfaces responsivas.
