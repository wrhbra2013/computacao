Carga Horária: 400h

Infraestrutura para Sistemas de Software COM31080h

Compreender e aplicar conceitos de comunicação através de redes de computadores e forma de organização de sistemas em nuvem.

Ementa: Redes de computadores e a internet. Noções de redes TCP/IP. Máquinas virtuais. Computação em nuvem. Contêineres e desenvolvimento usando contêineres. Noções de uso do docker e do openshift.

Vídeos

Bibliografia

  • TANENBAUM, A. S.; J. WETHERALL, D. J. Redes de Computadores. Pearson, 5ª Ed., 2011.
  • Vitalino, Jeferson Fernando Noronha; Castro, Marcus André Nunes. Descomplicando o Docker. Brassport, 2018.

Plataforma de Ingestão e Análise de Dados COM42080h

Desenvolver conceitos e práticas para coleta e análise de dados em sistemas de internet das coisas.

Ementa: Nuvens computacionais: características essenciais, modelos de serviço. Integração de dispositivos IoT a nuvens computacionais. Soluções de armazenamento de dados. Plataformas para IoT (AWS, Azure, IBM Bluemix).

Vídeos

Bibliografia

  • TANENBAUM, A. S.; J. WETHERALL, D. J. Redes de Computadores. Pearson, 5ª Ed., 2011.
  • Stallings, William. Arquitetura e organização de computadores. Pearson, 2017.

Desenvolvimento Web COM32080h

Desenvolver sistemas de gestão de informação e de conteúdo através de aplicações Web.

Ementa: Noções de escalabilidade e concorrência. Arquiteturas Web (3 camadas, MVC). Tecnologias de Interface Gráfica com Usuário baseada em Javascript (AngularJS, Vue.js, React). Frameworks back-end (Django). Noções de segurança da informação.

Vídeos

Bibliografia

  • MILETTO, Evandro Manara; BERTAGNOLLI, Silvia de Castro. Desenvolvimento de Software II. Bookman, 2014.
  • Santos, Valquiria. Projeto de interface com o usuário. Pearson, 2017.

Interface Humano-Computador COM43080h

Desenvolvimento de interfaces de usuário com ênfase em WEB com noções de acessibilidade.

Ementa: Histórico e evolução de IHC. Métodos e técnicas de design. Experiência de usuário, arquitetura da informação e acessibilidade. Páginas responsivas. Manipulação do DOM com JavaScript. REST e AJAX.

Vídeos

Bibliografia

  • Rocha, H.V.; Baranauskas, M.C.C. Design e Avaliação de Interfaces Humano-Computador. NIED, 2003.

Projeto Integrador III PGP00380h

Desenvolver um software com framework web ou aplicativo que utilize banco de dados, inclua script web, nuvem, acessibilidade, controle de versão, integração contínua e testes.

Ementa: Resolução de problemas. Levantamento de requisitos. Desenvolvimento web com framework. HTML, CSS. Linguagem de script. Banco de Dados. Nuvem. API. Acessibilidade. Integração Contínua. Testes.
