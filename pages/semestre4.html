4º Semestre

Carga Horária: 400h

Banco de Dados COM30080h

Projetar e implementar bancos de dados.

Ementa: Introdução a banco de dados. Modelagem de dados (modelo entidade-relacionamento). Linguagem SQL. Indexação. Bancos de Dados NoSQL. Mapeamento objeto-relacional (ORM).

Vídeos

Bibliografia

  • Elmasri, Ramez; Navathe, Shamkant B. Sistema de Banco de Dados. Pearson, 2018.
  • DATE, C. J. Introdução a Sistemas de Bancos de Dados. Elsevier, 2003.

Cálculo II MCA00280h

Ampliar a extensão de cálculo de uma variável para várias variáveis. Apresentar conceitos e técnicas que permitem a solução de problemas práticos em várias disciplinas.

Ementa: Funções de Várias Variáveis Reais. Fórmula de Taylor. Máximos e Mínimos. Integrais Múltiplas. Integrais de Linha. Teorema da Divergência. Teorema de Stokes.

Vídeos

Bibliografia

  • ÁVILA. G. Cálculo das funções de múltiplas variáveis. LTC, 2006.
  • GUIDORIZZI, H. L. Um curso de cálculo: volume 3. LTC, 2002.

Circuitos Digitais ELE30080h

Introduzir os conceitos dos componentes lógicos básicos e a forma de descreve-los utilizando uma linguagem de descrição de hardware (HDL).

Ementa: Introdução aos conceitos básicos de projeto lógico. Portas lógicas. Minimização de funções lógicas. Circuitos combinacionais. Elementos de memória: Latch, Flip-Flops, contadores. Introdução à HDL.

Vídeos

Bibliografia

  • VAHID, F. Sistemas Digitais: Projeto, Otimização e HDLs. Bookman, 2008.
  • Tocci, Ronald J.; Widmer, Neal S.; Moss, Gregory L. Sistemas digitais princípios e aplicações. Pearson, 2018.

Física do Movimento FIS30080h

Introduzir os conceitos fundamentais da mecânica clássica bem como as leis que regem os movimentos de pontos materiais.

Ementa: Espaço Tempo e Matéria. Cinemática Escalar e Vetorial. Vetores. Forças e Leis de Newton. Aplicações das Leis de Newton. Energia. Sistemas de Partículas. Colisões. Gravitação.

Vídeos

Bibliografia

  • HALLIDAY, D.; RESNICK, R.; WALKER, J. Fundamentos de física 1: mecânica. LTC, 2012.
  • NUSSENZVEIG, H. M. Curso de física básica: volume 1. Blucher, 2013.

Projeto Integrador I PGP10180h

Desenvolver um software com framework web que utilize noções de banco de dados, praticando controle de versão.

Ementa: Resolução de problemas. Levantamento de requisitos. Desenvolvimento web com framework. HTML. CSS. Banco de Dados. Controle de Versão.
